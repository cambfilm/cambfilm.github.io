Vending Machine

Our first capstone involved the creation of a vending machine command line interface. Users are given a list of items to purchase after "feeding money" into the program.

The stock list is generated by a text file that the program reads in using a file input system. The items decrease in stock as the user selects them. A count of items is displayed to user and indicates whether or not an item is sold out.

After a user is finished purchasing items, their change is displayed in coins (only the necessary amount) and a list of snacking noises is printed out corresponding to the treats their consumed.
