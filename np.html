Reservation CLI

The second capstone at Tech Elevator involved creating a reservation system that would allow a user to search a database of campsites and book a stay.

Users are able to give the program a range of dates and are presented with a list of campsites that are available to reserve. They can then chose a campsite and their reservation is booked in the database.

Creating this program allowed me to fully realize the possilities of creating relationships like this with a database.
